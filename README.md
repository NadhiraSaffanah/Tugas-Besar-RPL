# Tugas-Besar-RPL
Creating assignment management website
Tujuan Tugas Besar: Menciptakan sebuah sistem manajemen tugas besar.

Bahasa Pemrograman: Java
IDE: VSCode
Database: PostGre
Software untuk track project beserta keterangan jobdesk masing-masing tim: Trello (Kanban Board)

Cara melakukan upload pada Github:
1. Buat branch untuk masing-masing fitur.
2. Pull file dari main.
3. Kerjakan masing-masing jobdesk sesuai jadwal.
4. Push hasil pekerjaan ke branch pribadi terlebih dahulu.
5. Ajukan request merge ke branch main.
6. Request akan di-review oleh PM, lalu diskusi jika request kurang benar.
7. Jika reqeust diterima maka file yang sudah dikerjakan tim sudah berhasil digabungkan di main.
8. Setelah sudah di-merge, anggota tim harus melakukan pull kembali dari main.

NOTES:
1. Gunakan nama branch yang deskriptif.
2. Tentukan deskripsi commit dengan jelas.
3. Gunakan Bahasa Indonesia untuk commit dan lainnya.
